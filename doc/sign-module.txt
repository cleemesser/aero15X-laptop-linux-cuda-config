The ubuntu wiki has info on DKMS with dkms
https://wiki.ubuntu.com/UEFI/SecureBoot/DKMS

General info on building modules:
https://www.kernel.org/doc/html/latest/kbuild/modules.html

can create the signing key once with create-uefi-keys.sh 


Install the signing key for the current kernel
```
cd /lib/modules/`uname -r`/build/certs
sudo cp ~clee/.ssl/MOK.pem signing_key.pem
```
